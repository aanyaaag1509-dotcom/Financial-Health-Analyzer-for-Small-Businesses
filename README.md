# Financial-Health-Analyzer-for-Small-Businesses
## Project Overview

The Financial Health Analyzer is an Excel-based financial analysis project developed to assess the financial performance of a small retail business using accounting statements, financial ratios, and a custom Financial Health Score Model. The project converts raw financial data into meaningful insights through an interactive dashboard.
## Objectives

- Prepare financial statements.
- Analyze business performance.
- Calculate key financial ratios.
- Develop a Financial Health Score Model.
- Build an interactive Excel dashboard.
- Generate actionable business insights.
